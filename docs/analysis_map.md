# Analysis Map

This file points readers from common manuscript claims to the public scripts that reproduce them.

| Result or check | Script location |
| --- | --- |
| Constructiveness/destructiveness correlations and discriminant validity | `analysis/discriminant_validity/` |
| Overall prevalence and feature prevalence | `analysis/prevalence/` |
| Surfacing, likes, replies, and engagement models | `analysis/engagement/` |
| Parent-child reply propagation | `analysis/propagation/` |
| Agreement/disagreement robustness | `analysis/agreement_robustness/` |
| Stance robustness | `analysis/stance_robustness/` |
| Deleted-comment and temporal checks | `analysis/temporal_and_deleted_comments/` |
| Climate-change replication | `analysis/climate_replication/` |
| Main figures and supporting figure files | `analysis/figures/`; `outputs/figures/` |
| Supplemental model specifications and diagnostics | `analysis/supplement_*` |
| Human validation and calibration summaries | `validation/scripts/`; `validation/tables/` |
